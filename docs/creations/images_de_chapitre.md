# Images de chapitre

Voici une ressource créée par Bonux, qui permet d'utiliser des images de sélection de chapitres, pour la création des histoires.

![](assets/img/images_de_chapitre.png)

Dans l'archive ZIP, il y a des images pour les chapitres 1 a 40, ainsi qu'un PSD et XCF si vous voulez personnaliser.
